âš™ï¸ How It Works
 * The Prompt: User enters a descriptive prompt (e.g., "A neon-lit cyberpunk cat in a rainy Tokyo street").
 * The Magic: The frontend calls the generateImages() method from the Firebase AI Logic SDK.
 * The Storage: The resulting image data is converted to a blob and uploaded to a dedicated folder in Firebase Storage.
 * The Record: A new document is added to the images collection in Firestore, containing the Storage URL and the prompt.
 * The Gallery: The UI listens to Firestore changes and updates the masonry grid in real-time.

1. Clone the repository
git clone https://github.com/your-username/dream-gallery.git
cd dream-gallery

2. Setup Firebase
 * Create a project in the Firebase Console.
 * Enable Firestore, Storage, and Firebase AI Logic.
 * Register your Web App and copy the configuration.
3. Environment Variables
Create a .env file in the root directory:
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com

4. Install & Run
npm install
npm run dev
